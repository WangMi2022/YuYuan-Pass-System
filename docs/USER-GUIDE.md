# YuYuan Pass System 用户使用文档

## 1. 文档说明

本文面向系统管理员、资产管理员和普通业务用户，说明系统登录、资产管理、文档协作、站点收藏、公告提醒以及系统配置的日常使用方法。

## 2. 登录与工作台

### 2.1 登录

1. 在浏览器打开系统地址。
2. 输入用户名和密码。
3. 根据页面要求输入验证码。
4. 点击“登录”进入系统。

若账号被冻结、密码错误或角色未分配权限，系统会给出相应提示。连续登录失败时请联系管理员处理，不要重复尝试未知密码。

桌面端登录页左侧用于完整展示背景图片，登录表单固定在右侧；移动端自动调整为上方背景、下方表单。管理员配置的登录图标和背景会在刷新登录页后生效。

### 2.2 页面结构

- 左侧菜单：当前分为首页驾驶舱、资产管理、协同办公和系统管理四个业务入口。
- 顶部标签：保留已经打开的业务页面，可快速切换或关闭。
- 顶部搜索：搜索菜单和系统功能。
- 公告铃铛：查看未读公告和公告详情。
- 主题按钮：切换亮色与暗色模式。
- 用户菜单：查看个人信息、切换角色或退出系统。
- 侧边栏提示卡：在展开侧边栏的左下方显示资产管理提示；侧边栏折叠、移动端或窗口高度不足时会自动隐藏。

## 3. 资产管理

### 3.1 资产可视化大屏

菜单路径：`首页驾驶舱`。

大屏展示：

- 资产实物总量与档案数量。
- 资产分类数量。
- 资产原值、当前估值和累计价值减少。
- 分类资产价值图表和自动滚动明细。
- 使用中、闲置、维修中和已处置状态构成。
- 资产位置数量排行。
- 最近登记资产。

点击“刷新”可立即重新获取统计数据；点击“全屏展示”可进入大屏模式。

### 3.2 资产分类

菜单路径：`资产管理 → 分类管理`。

#### 新增分类

1. 点击“新增分类”。
2. 填写分类名称和唯一分类编码。
3. 选择分类标识颜色。
4. 填写分类说明。
5. 保存。

已有资产引用的分类不能直接删除，应先调整对应资产的分类。

### 3.3 资产档案

菜单路径：`资产管理 → 资产档案`。

#### 新增资产

1. 点击“新增资产”。
2. 填写资产编号、名称和分类。
3. 补充品牌、型号、序列号、数量和单位。
4. 填写采购单价和当前估值。
5. 选择资产状态、位置和保管人。
6. 根据需要上传资产照片并填写备注。
7. 保存资产。

资产原值按照“数量 × 采购单价”计算。当前估值用于管理视角下的价值展示，不替代正式财务折旧和会计凭证。

#### 查询资产

可以按关键词、分类、状态和位置筛选。关键词会匹配资产名称、编号、品牌、型号等信息。

#### 修改和删除

- “编辑”用于修改资产档案。
- “删除”会移除资产记录，操作前系统会要求确认。
- 重要数据删除前应先完成备份或导出。

## 4. 文档管理

菜单路径：`协同办公 → 文档管理`。

### 4.1 支持格式

- Markdown：`.md`
- 文本与结构化文本：`.txt`、`.json`、`.csv`、`.yaml`、`.html`
- Word：`.doc`、`.docx`、`.docs`
- Excel：`.xls`、`.xlsx`
- PDF：`.pdf`

不同格式的预览和编辑能力存在差异。旧版二进制 Word 格式可能只能下载，无法完整在线预览。

### 4.2 上传文档

1. 进入文档查看页面。
2. 点击上传按钮。
3. 选择本地文档。
4. 等待上传完成。

源文件保存到对象存储，列表中会显示文件名、格式、大小和更新时间。

### 4.3 查看源文件

点击文档名称后，系统默认打开“原文件预览”：

- Word、Excel、PDF 使用对应预览组件展示。
- Markdown 和文本文件以只读内容展示。
- 不支持浏览器直接预览的格式可以下载源文件。

### 4.4 在线编辑

1. 打开目标文档。
2. 点击“开始编辑”。
3. 在独立编辑区域修改内容。
4. 点击“保存在线版本”。

注意事项：

- 在线版本与 OSS 中的原始文件分开保存。
- 保存在线版本不会覆盖原始 Word 或 Excel 文件。
- Excel 普通查看状态以数据表格为主，开始编辑后进入独立编辑框。
- 编辑大型 Excel 文件时，浏览器解析可能需要一定时间。

## 5. 媒体库

菜单路径：`协同办公 → 媒体库`。

媒体库仅允许上传图片，用于资产照片、头像、登录背景和富文本插图。普通上传和压缩上传均会写入已配置的 OSS/S3 存储。

建议优先使用 JPG、PNG 或 WebP，避免上传尺寸过大或内容重复的图片。

## 6. 站点管理

菜单路径：`协同办公 → 站点收藏`。

### 6.1 新增站点

1. 点击“新增站点”。
2. 填写站点名称和 HTTP/HTTPS 地址。
3. 设置分类、标识颜色、排序和启用状态。
4. 填写用途说明。
5. 保存。

点击卡片或“打开”按钮会在新窗口访问站点，并累计访问次数。

### 6.2 维护站点

- 按名称、地址、分类或说明搜索。
- 按分类和启用状态筛选。
- 停用站点不会被打开，但记录仍会保留。
- 删除操作需要确认。

## 7. 公告提醒

### 7.1 查看公告

顶部铃铛上的数字表示未读公告数量。点击铃铛可查看最近公告，点击公告标题打开详情。

查看详情后该公告会自动标记为已读；“全部已读”可一次清除当前用户的所有未读状态。

### 7.2 发布公告

菜单路径：`协同办公 → 公告管理`。

1. 点击“新增”。
2. 填写公告标题和内容。
3. 根据需要添加附件。
4. 点击“保存草稿”暂不发布，或点击“发布公告”立即发布。

发布后：

- 在线用户通过 SSE 立即收到通知。
- 离线用户下次登录后看到未读标记。
- 已读状态按用户保存，可跨设备同步。

## 8. 系统设置

菜单路径：`系统管理 → 系统配置`。

### 8.1 更换登录图标

1. 查看当前登录图标预览。
2. 点击“上传并应用”。
3. 选择 JPG、PNG 或 WebP 图片。
4. 等待图片上传至 OSS 并自动应用。

建议使用内容居中、背景透明或纯色的正方形图片。单张图标不能超过 2 MB。点击“恢复默认”可重新启用系统内置图标。

自定义图标会同时显示在登录表单顶部和登录页品牌区域。

### 8.2 更换登录背景

1. 查看当前登录背景缩略图。
2. 点击“变更管理”。
3. 上传 JPG、PNG 或 WebP 图片。
4. 在背景图库中选择目标缩略图。
5. 点击“保存并应用”。

上传图片只会加入图库，必须选择并保存后才会切换登录页背景。正在使用的背景不能删除。

建议桌面端背景使用宽屏图片，并将重要主体安排在画面中部或左侧，避免不同屏幕裁切时丢失关键内容。登录表单位于独立的右侧区域，不会覆盖背景图。

## 9. 用户、角色与权限

### 9.1 用户管理

管理员可以新增、编辑、冻结用户，设置角色和重置密码。

### 9.2 角色权限

角色权限由菜单权限、API 权限和按钮权限组成。新建非管理员角色后，应同时分配业务菜单及其对应 API，否则可能出现菜单可见但接口无权访问的情况。

### 9.3 操作记录

菜单路径：`系统管理 → 操作审计`。

系统会记录关键新增、修改、删除和配置操作，管理员可以按请求路径、状态和时间进行审计。

## 10. 常见问题

### 页面白屏或动态模块加载失败

1. 使用 `Ctrl + F5` 强制刷新。
2. 清理浏览器站点缓存。
3. 确认 Web 容器已经完成最新版本部署。
4. 检查缺失的 `/assets/*.js` 是否返回 404，而不是 HTML。

### 图片加载失败

- 确认对象存储服务和桶可访问。
- 检查桶策略或公开读取策略。
- 检查系统配置中的 endpoint、bucket 和 bucket URL。

### 看不到菜单

- 退出后重新登录以刷新动态菜单。
- 检查当前角色是否具有菜单权限和 API 权限。

### 收不到实时公告

- 确认浏览器到 `/api/info/stream` 的连接未被代理中断。
- 检查公告状态是否为“已发布”。
- 即使 SSE 暂时断开，系统仍会定时刷新未读公告。

### Excel 编辑卡顿

- 大文件首次解析需要时间。
- 优先在源文件预览中查看，确认需要修改后再点击“开始编辑”。
- 建议减少无用空白行、超大图片和复杂合并单元格。

## 11. 退出系统

点击右上角用户头像，在用户菜单中选择“登出”。公共或共享电脑使用完毕后必须退出账号并关闭浏览器。
